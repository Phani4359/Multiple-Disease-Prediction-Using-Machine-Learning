# Multiple Disease Prediction Using Machine Learning

## Overview

This project is a Machine Learning-based application that predicts the likelihood of multiple diseases using patient health data. The system helps users analyze medical information and provides predictions for different diseases using trained ML models.

## Diseases Predicted

- Diabetes
- Heart Disease
- Parkinson's Disease

## Features

- Predicts multiple diseases from patient input.
- Data preprocessing and feature scaling.
- Model training and evaluation.
- Performance comparison of Machine Learning algorithms.
- Easy-to-use Google Colab notebook.

## Technologies Used

- Python
- Google Colab
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- TensorFlow/Keras (if used)

## Machine Learning Models

- Random Forest
- Decision Tree
- Neural Network

## Dataset

The dataset contains patient medical information such as:

- Age
- Gender
- BMI
- Blood Pressure
- Glucose
- Cholesterol
- Heart Rate
- Smoking Status
- Exercise Level
- Family History

## Workflow

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Data Preprocessing
5. Model Training
6. Model Evaluation
7. Disease Prediction

## Installation

Clone the repository

```bash
git clone https://github.com/Phani4359/Multiple-Disease-Prediction-Using-Machine-Learning.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the notebook

```bash
jupyter notebook
```

or open it directly in Google Colab.

## Results

The trained models achieved good prediction performance on the test dataset. Model accuracy and evaluation metrics can be found in the notebook.
